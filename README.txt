README for Java MyCareNet Project
=================================

=== 1. Introduction

This project contains the source code tree of Java MyCareNet Project.
The source code is hosted at: https://code.google.com/p/mycarenet/


=== 2. Requirements

The following is required for compiling the software:
* Oracle Java 1.7.0_71
* Apache Maven 3.2.3


=== 3. Build

The project can be build via:
	mvn clean install


=== 4. Release

Release the project via:
	mvn release:prepare
	mvn release:perform
